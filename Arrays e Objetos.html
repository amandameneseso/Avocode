Arrays e Objetos

Arrays e objetos são dois dos tipos mais comuns de estrutura de dados. Vimos que array é um tipo de objeto e que eles são usados para armazenar e organizar dados, mas funcionam de maneiras diferentes e têm características distintas. Vamos ver as diferenças e como usá-los.

Array

Um array é uma estrutura de dados usada para armazenar múltiplos valores em uma única variável. Podem conter qualquer tipo de dado (números, strings, objetos, etc.). Com arrays, podemos armazenar dados ordenados e acessar os elementos pelo índice. Exemplo:

let frutas = ["banana", "macã", "laranja"];

Nota: Arrays são um tipo especial de objeto. O operador typeof retorna "objeto" para arrays:

console.log(typeof frutas); // object

Por que usar arrays?

Se você tiver uma lista de itens (uma lista de nomes de frutas, por exemplo), armazenar as frutas em variáveis individuais pode se parecer com isso:

let fruta1 = "maçã";
let fruta2 = "banana";
let fruta3 = "laranja";

No entanto, e se você quiser percorrer as frutas e encontrar uma específica? E se você não tivesse 3 frutas, mas 300? Nesses casos, usar arrays pode ser muito mais eficiente.

Um array pode conter muitos valores sob um único nome, e você pode acessar os valores referindo-se a um número de índice.

Declarando um array

Para declarar um array, você pode usar colchetes [ ] (array literal) ou o construtor new Array():

const frutas = ["maçã", "banana", "laranja"];
const frutas = new Array("maçã", "banana", "laranja");

Os dois exemplos acima fazem exatamente o mesmo. Para simplicidade, legibilidade e velocidade de execução, use o método literal de array.

Acessando elementos do array

Os elementos de um array são indexados começando do zero. Para acessar um elemento, usa-se o nome do array seguido do número do índice. Exemplo:

const frutas = ["maçã", "banana", "laranja"];
console.log(frutas[0]); // maçã
console.log(frutas[1]); // banana
console.log(frutas[2]); // laranja

Alterando um elemento do array

Esta declaração altera o valor do primeiro elemento em frutas:

frutas[0] = "abacaxi";
console.log(frutas[0]); // abacaxi

Nesse exemplo, como o primeiro elemento do array foi alterado, frutas[0] retorna abacaxi.

Métodos comuns para manipular arrays

push() adiciona um elemento ao final do array:

frutas.push("uva");
console.log(frutas); // ["maçã", "banana", "laranja", "uva"]

pop() remove o último elemento do array:

frutas.pop();
console.log(frutas); // ["maçã", "banana", "laranja"]

shift() remove o primeiro elemento do array:

frutas.shift();
console.log(frutas); // ["banana", "laranja"]

unshift() adiciona um elemento ao inicio do array:

frutas.unshift("manga");
console.log(frutas); // ["manga", "banana", "laranja"]

length retorna o número de elementos no array:

console.log(frutas.length); // 3

Nota: Como o índice de um array começa em 0, length sempre retorna um valor uma unidade maior que o índice do último elemento. Dessa forma, o índice do último elemento de um array é sempre length - 1.

join() retorna uma string com todos os elementos do array concatenados, separando-os com um delimitador especificado entre as aspas (opcional):

console.log(frutas.join(", ")); // manga, banana, laranja

sort() ordena os elementos em ordem alfabética:

frutas.sort();
console.log(frutas); // ["banana", "laranja", "manga"]

reverse() inverte a ordem dos elementos do array:

frutas.reverse();
console.log(frutas); // ["manga", "laranja", "banana"]

Como ordenar arrays de números?

Temos o seguinte array:

let num = [25, 100, 5, 60, 40];

Ao ordenar esse array utilizando num.sort(), o resultado será:

num.sort();
console.log(num); // [100, 25, 40, 5, 60]

Isso ocorre porque, por padrão, o método sort() classifica os valores como strings. Se os números forem classificados como strings, "25" é maior que "100", porque "2" é maior que "1". Objeto

Objetos na vida real

Na vida real, objetos são coisas como: casas, carros, pessoas, animais ou quaisquer outros objetos. Objetos possuem:

  • Propriedades: características que descrevem o objeto.
  • Valores: os dados associados a essas características.

Por exemplo, um objeto que representa uma casa pode ter:

  • cor: amarelo
  • tamanho: 100m2
  • preço: 200.000

Objetos de casa têm as mesmas propriedades, mas os valores diferem de casa para casa.

Objetos em JavaScript

Um objeto é uma variável especial que pode conter vários valores organizados como pares chave-valor. As chaves são como os nomes das propriedades, e os valores são os dados correspondentes.

Sintaxe básica:

Um objeto é definido usando {} (chaves) e os pares chave-valor separados por : (dois pontos). Os pares são separados por vírgulas.

const casa = {
    cor: "amarelo",
    tamanho: 100,
    preco: 200000
};

Acessando propriedades

Você pode acessar propriedades de objetos de duas maneiras:

1. Notação de ponto (.):

console.log(casa.cor); // amarelo
console.log(casa.tamanho); // 100

1. Notação de colchetes ([]):

console.log(casa["cor"]); // amarelo
console.log(casa["tamanho"]); // 100
