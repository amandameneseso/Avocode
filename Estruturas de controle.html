Estruturas de controle

As estruturas de controle permitem gerenciar o fluxo de execução de um programa, seja para tomar decisões, repetir ações ou gerenciar o fluxo de execução do programa com base em condições ou padrões específicos. Elas são essenciais para criar lógicas dinâmicas e flexíveis. e gerenciar o fluxo de execução do programa com base em condições ou padrões específicos. Vamos explorar as principais categorias e exemplos de cada uma:

Estruturas condicionais

If, else e else if

Muitas vezes, quando você escreve um código, você quer executar ações diferentes para decisões diferentes. Você pode usar estruturas condicionais para fazer isso. Em JavaScript, temos as seguintes estruturas condicionais:

  • if: Executa uma ação se uma condição for verdadeira.
  • else: Executa uma ação se a mesma condição for falsa.
  • else if: Especificar uma nova condição a ser testada, se a primeira condição for falsa. Permite a construção de múltiplas condições em sequência.
  • switch: Executa uma ação de acordo com o valor de uma expressão.

As estruturas condicionais estão ligadas à tomada de decisão de um algoritmo. Ao utilizar expressões que retornam verdadeiro ou falso, o algoritmo executa o bloco de comandos relativos a este resultado.

Primeiramente, precisamos entender a sintaxe:

if (condição) {
   // Código a ser executado se a condição for verdadeira
} else if (outraCondição) {
   // Código a ser executado se a primeira condição for falsa.
} else {
   // Código a ser executado se nenhuma condição for verdadeira
}

Agora, podemos aplicar em um exemplo:

if (hora < 18) {
   saudação = "Bom dia!";
} else {
   saudação = "Boa noite!";
}

Usando o else if para especificar uma nova condição se a primeira condição for falsa:

if (hora < 12) {
   saudação = "Bom dia!";
} else if (hora < 18) {
   saudação = "Boa tarde!";
} else {
   saudação = "Boa noite!";
}

No exemplo acima, if, else if e else Verificam:

  • Se a hora é menor que 12, a saudação será "Bom dia!"
  • Se a hora é menor que 18, a saudação será "Boa tarde!"
  • Caso contrário, será "Boa noite!"

Vejamos outro exemplo:

if (idade > 18) {
   console.log("Você é obrigado a votar.");
}

Nesse exemplo, o if verifica se o valor da variável idade é maior que 18. Se a condição for verdadeira, o código dentro do bloco será executado. Caso contrário, o bloco será ignorado (nenhuma saída será exibida).

Porém este exemplo não está completo, afinal, há quem não possa votar e também a possibilidade de voto opcional. Podemos utilizar os operadores lógicos na condição para deixar o exemplo mais robusto. Para isso, usamos os operadores lógicos && (e) e || (ou):

if (idade >= 18 && idade < 70) {
   console.log("Você é obrigado a votar.");
} else if (idade >= 16 || idade >= 70) {
   console.log("Seu voto é opcional.");
} else {
   console.log("Você não pode votar.");
}

Nesse exemplo, o if verifica se a idade está entre 18 e 69 anos, o que significa que a pessoa é obrigada a votar. O else if, que iniciará caso a primeira condição seja falsa, verifica duas condições: se a idade é maior ou igual a 16 ou se a idade é maior ou igual a 70 anos. Para essas faixas etárias, o voto é opcional. Caso a idade seja menor que 16 anos, a pessoa não pode votar.

Operador ternário

É uma maneira curta de escrever uma estrutura condicional if-else. Ele possui a seguinte sintaxe:

condição ? valor_se_verdadeiro : valor_se_falso;

Explicação:

  • condição: A expressão que será avaliada (verdadeira ou falsa).
  • valor_se_verdadeiro: O valor ou expressão a ser retornada se a condição for verdadeira.
  • valor_se_falso: O valor ou expressão a ser retornada se a condição for falsa.

Exemplo:

let numero = 7;
let resultado = numero % 2 === 0 ? "O número é par." : "O número é ímpar.";
console.log(resultado); // "O número é ímpar."

Nesse exemplo, numero % 2 === 0 verifica se o número é divisível por 2 (se o resto da divisão for 0, significa que é par). Se a condição for verdadeira, o valor "O número é par." é retornado. Se a condição for falsa, o valor "O número é ímpar." é retornado.

Este exemplo escreverá um link para a página inicial deste site ou para o World Wildlife Fund (WWF). Usando uma estrutura condicional e um número aleatório, há uma chance de 50% para cada um dos links.

let text;
if (Math.random() < 0.5) {
   text = "<a href='index.html'>Link aleatório</a>";
} else {
   text = "<a href='https://wwf.org'>Link aleatório</a>";
}
document.getElementById("demo").innerHTML = text;

Switch

Outra estrutura condicional do JavaScript que podemos utilizar é o Switch Case. O Swith Case é uma instrução que se comporta de forma semelhante ao if / else, porém possui uma estrutura mais organizada e de fácil compreensão. Lembrando que só é aceito valores pré-definidos e não expressões condicionais.

O Switch Case escolhe entre várias opções com base no valor de uma variável ou expressão.

Sintaxe:

switch (expressão) {
   case valor1:
      // Código a ser executado se expressão === valor1
      break;
   case valor2:
      // Código a ser executado se expressão === valor2
      break;
   case valor3:
      // Código a ser executado se expressão === valor3
      break;
   default:
      // Código a ser executado se nenhum caso corresponder
}

Exemplo:

let dia = "segunda";
switch (dia) {
   case "segunda":
      console.log("Hoje é segunda-feira.");
      break;
   case "sexta":
      console.log("Hoje é sexta-feira.");
      break;
   default:
      console.log("Não é segunda nem sexta.");
}

No exemplo acima, o switch escolhe entre duas opções, de acordo com o valor de uma variável ou expressão. Ele verifica o valor da variável dia e imprime uma mensagem dependendo do dia da semana:

  • Se dia for igual a "segunda", a mensagem "Hoje é segunda-feira." será impressa.
  • Se dia for igual a "sexta", a mensagem "Hoje é sexta-feira." será impressa.
  • O default é usado para capturar qualquer valor que não seja "segunda" nem "sexta". Nesse caso, será impressa a mensagem "Não é segunda nem sexta.".

Como dia está definido como "segunda", o bloco de código associado ao case "segunda" será executado e a seguinte mensagem será exibida: "Hoje é segunda-feira."

Nota: Quando o JavaScript chega em um "break", ele sai do bloco de switch. Isso interromperá a execução dentro do respectivo bloco e continuará para o bloco seguinte. Não é necessário colocar "break" no último bloco. O bloco quebra (termina) lá de qualquer maneira.

Estruturas de repetição

Estruturas de repetição são úteis quando precisamos repetir várias vezes a execução de um bloco de comandos até que uma condição seja atendida. Elas eliminam a necessidade de escrever o mesmo código várias vezes, automatizando tarefas repetitivas.

Por exemplo, para imprimir os números de 1 a 10, em vez de escrever:

console.log(1);
console.log(2);
console.log(3);
console.log(4);
console.log(5);
console.log(6);
console.log(7);
console.log(8);
console.log(9);
console.log(10);

Podemos escrever:

for (let i = 1; i <= 10; i++) {
   console.log(i);
}

JavaScript suporta diferentes tipos de loops:

  • for: Contar ou iterar um número específico de vezes.
  • for/in: Iterar sobre as chaves de um objeto ou índices de um array.
  • for/of: Iterar sobre os valores de objetos iteráveis
  • while: Repetir enquanto uma condição for verdadeira
  • do/while: Garantir pelo menos uma execução antes de verificar a condição.
  • forEach: Executar uma função para cada elemento de um array.

Iterar significa repetir ou percorrer (passar por) um conjunto de elementos, geralmente de uma estrutura de dados (como um array, objeto ou outro tipo de coleção), de maneira sequencial para executar uma ação. Existem várias formas de fazer isso em JavaScript, e cada método possui características específicas que podem ser úteis em diferentes situações.
