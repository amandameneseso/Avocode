Estruturas de controle

As estruturas de controle permitem gerenciar o fluxo de execução de um programa, seja para tomar decisões, repetir ações ou gerenciar o fluxo de execução do programa com base em condições ou padrões específicos. Elas são essenciais para criar lógicas dinâmicas e flexíveis. e gerenciar o fluxo de execução do programa com base em condições ou padrões específicos. Vamos explorar as principais categorias e exemplos de cada uma:

Estruturas condicionais

If, else e else if

Muitas vezes, quando você escreve um código, você quer executar ações diferentes para decisões diferentes. Você pode usar estruturas condicionais para fazer isso. Em JavaScript, temos as seguintes estruturas condicionais:

  • if: Executa uma ação se uma condição for verdadeira.
  • else: Executa uma ação se a mesma condição for falsa.
  • else if: Especificar uma nova condição a ser testada, se a primeira condição for falsa. Permite a construção de múltiplas condições em sequência.
  • switch: Executa uma ação de acordo com o valor de uma expressão.

As estruturas condicionais estão ligadas à tomada de decisão de um algoritmo. Ao utilizar expressões que retornam verdadeiro ou falso, o algoritmo executa o bloco de comandos relativos a este resultado.

Primeiramente, precisamos entender a sintaxe:

if (condição) {
   // Código a ser executado se a condição for verdadeira
} else if (outraCondição) {
   // Código a ser executado se a primeira condição for falsa.
} else {
   // Código a ser executado se nenhuma condição for verdadeira
}

Agora, podemos aplicar em um exemplo:

if (hora < 18) {
   saudação = "Bom dia!";
} else {
   saudação = "Boa noite!";
}

Usando o else if para especificar uma nova condição se a primeira condição for falsa:

if (hora < 12) {
   saudação = "Bom dia!";
} else if (hora < 18) {
   saudação = "Boa tarde!";
} else {
   saudação = "Boa noite!";
}

No exemplo acima, if, else if e else Verificam:

  • Se a hora é menor que 12, a saudação será "Bom dia!"
  • Se a hora é menor que 18, a saudação será "Boa tarde!"
  • Caso contrário, será "Boa noite!"

Vejamos outro exemplo:

if (idade > 18) {
   console.log("Você é obrigado a votar.");
}

Nesse exemplo, o if verifica se o valor da variável idade é maior que 18. Se a condição for verdadeira, o código dentro do bloco será executado. Caso contrário, o bloco será ignorado (nenhuma saída será exibida).

Porém este exemplo não está completo, afinal, há quem não possa votar e também a possibilidade de voto opcional. Podemos utilizar os operadores lógicos na condição para deixar o exemplo mais robusto. Para isso, usamos os operadores lógicos && (e) e || (ou):

if (idade >= 18 && idade < 70) {
   console.log("Você é obrigado a votar.");
} else if (idade >= 16 || idade >= 70) {
   console.log("Seu voto é opcional.");
} else {
   console.log("Você não pode votar.");
}

Nesse exemplo, o if verifica se a idade está entre 18 e 69 anos, o que significa que a pessoa é obrigada a votar. O else if, que iniciará caso a primeira condição seja falsa, verifica duas condições: se a idade é maior ou igual a 16 ou se a idade é maior ou igual a 70 anos. Para essas faixas etárias, o voto é opcional. Caso a idade seja menor que 16 anos, a pessoa não pode votar.

Operador ternário

É uma maneira curta de escrever uma estrutura condicional if-else. Ele possui a seguinte sintaxe:

condição ? valor_se_verdadeiro : valor_se_falso;

Explicação:

  • condição: A expressão que será avaliada (verdadeira ou falsa).
  • valor_se_verdadeiro: O valor ou expressão a ser retornada se a condição for verdadeira.
  • valor_se_falso: O valor ou expressão a ser retornada se a condição for falsa.

Exemplo:

let numero = 7;
let resultado = numero % 2 === 0 ? "O número é par." : "O número é ímpar.";
console.log(resultado); // "O número é ímpar."

Nesse exemplo, numero % 2 === 0 verifica se o número é divisível por 2 (se o resto da divisão for 0, significa que é par). Se a condição for verdadeira, o valor "O número é par." é retornado. Se a condição for falsa, o valor "O número é ímpar." é retornado.

Este exemplo escreverá um link para a página inicial deste site ou para o World Wildlife Fund (WWF). Usando uma estrutura condicional e um número aleatório, há uma chance de 50% para cada um dos links.

<p id="demo"></p>

let text;
if (Math.random() < 0.5) {
   text = "<a href='index.html'>Link aleatório</a>";
} else {
   text = "<a href='https://wwf.org'>Link aleatório</a>";
}
document.getElementById("demo").innerHTML = text;

Switch

Outra estrutura condicional do JavaScript que podemos utilizar é o Switch Case. O Swith Case é uma instrução que se comporta de forma semelhante ao if / else, porém possui uma estrutura mais organizada e de fácil compreensão. Lembrando que só é aceito valores pré-definidos e não expressões condicionais.

O Switch Case escolhe entre várias opções com base no valor de uma variável ou expressão.

Sintaxe:

switch (expressão) {
   case valor1:
      // Código a ser executado se expressão === valor1
      break;
   case valor2:
      // Código a ser executado se expressão === valor2
      break;
   case valor3:
      // Código a ser executado se expressão === valor3
      break;
   default:
      // Código a ser executado se nenhum caso corresponder
}

Exemplo:

let dia = "segunda";
switch (dia) {
   case "segunda":
      console.log("Hoje é segunda-feira.");
      break;
   case "sexta":
      console.log("Hoje é sexta-feira.");
      break;
   default:
      console.log("Não é segunda nem sexta.");
}

No exemplo acima, o switch escolhe entre duas opções, de acordo com o valor de uma variável ou expressão. Ele verifica o valor da variável dia e imprime uma mensagem dependendo do dia da semana:

  • Se dia for igual a "segunda", a mensagem "Hoje é segunda-feira." será impressa.
  • Se dia for igual a "sexta", a mensagem "Hoje é sexta-feira." será impressa.
  • O default é usado para capturar qualquer valor que não seja "segunda" nem "sexta". Nesse caso, será impressa a mensagem "Não é segunda nem sexta.".

Como dia está definido como "segunda", o bloco de código associado ao case "segunda" será executado e a seguinte mensagem será exibida: "Hoje é segunda-feira."

Nota: Quando o JavaScript chega em um "break", ele sai do bloco de switch. Isso interromperá a execução dentro do respectivo bloco e continuará para o bloco seguinte. Não é necessário colocar "break" no último bloco. O bloco quebra (termina) lá de qualquer maneira.

Estruturas de repetição

Estruturas de repetição são úteis quando precisamos repetir várias vezes a execução de um bloco de comandos até que uma condição seja atendida. Elas eliminam a necessidade de escrever o mesmo código várias vezes, automatizando tarefas repetitivas.

Por exemplo, para imprimir os números de 1 a 5, em vez de escrever:

console.log(1);
console.log(2);
console.log(3);
console.log(4);
console.log(5);

Podemos escrever:

for (let i = 1; i <= 5; i++) {
   console.log(i);
}

JavaScript suporta diferentes tipos de loops:

  • for: Contar ou iterar um número específico de vezes.
  • for/in: Iterar sobre as chaves de um objeto ou índices de um array.
  • for/of: Iterar sobre os valores de objetos iteráveis
  • while: Repetir enquanto uma condição for verdadeira
  • do/while: Garantir pelo menos uma execução antes de verificar a condição.

Iterar significa repetir ou percorrer (passar por) um conjunto de elementos, geralmente de uma estrutura de dados (como um array, objeto ou outro tipo de coleção), de maneira sequencial para executar uma ação. Existem várias formas de fazer isso em JavaScript, e cada método possui características específicas que podem ser úteis em diferentes situações. Veremos a seguir cada tipo de loop:

For

O loop for executa um bloco de código repetidamente, com base em uma condição especificada. Com ele, você controla exatamente o índice e o número de iterações.

Sua estrutura é: for (inicio; condição; incremento)

Componentes do for:

  • inicio: Define o valor inicial da variável de controle.
  • condição: Define a condição para que o loop seja executado
  • incremento: Aumenta um valor (i++) cada vez que o bloco de código no loop for executado (atualiza a variável de controle). Essa expressão pode fazer coisas como incremento negativo (i--), incremento positivo (i = i + 15), ou qualquer outra coisa.

Exemplo:

let frutas = ["maçã", "banana", "laranja"];

for (let i = 0; i < frutas.length; i++) {
   console.log(frutas[i]);
} // Saída: maçã
            banana
            laranja

Explicação:

  • A variável i começa em 0 (primeiro índice do array).
  • A condição i < frutas.length garante que o loop será executado enquanto n for menor que o comprimento do array.
  • A cada iteração, i aumenta em 1 (usando i++), e acessamos frutas[i], que retorna o elemento na posição i.

Exemplo 2:

for(let i = 0; i < 11; i++){
   console.log("5 x " + i + " = " + 5*i);
}

Ao executar o código, teremos o resultado:

5 x 0 = 0
5 x 1 = 5
5 x 2 = 10
5 x 3 = 15
5 x 4 = 20
5 x 5 = 25
5 x 6 = 30
5 x 7 = 35
5 x 8 = 40
5 x 9 = 45
5 x 10 = 50

For/in

Usado para percorrer todas as propriedades de um objeto. Ele é útil quando queremos acessar os nomes (ou chaves) das propriedades de um objeto e seus valores.

Sintaxe:

for (let chave in objeto) {
   // código a ser executado
}
  • chave: Representa o nome de cada propriedade (ou índice, no caso de arrays) do objeto sendo iterado.
  • objeto: O objeto cujas propriedades você deseja iterar.

Exemplo:

let personagem = {
   nome: "Aloy",
   idade: 20,
   tribo: "Nora"
};

for (let x in personagem) {
   console.log(x + ": " + personagem[x]);
} // Saída: nome: Aloy
            idade: 20
            tribo: Nora

Explicação:

  • O for...in percorre um objeto chamado personagem.
  • Cada iteração retorna uma chave (representada pela variável x).
  • Essa chave (x) é usada para acessar o valor da chave.
  • O valor associado à chave é personagem[x].

Nota: Ele também funciona com arrays, mas percorre os índices dos elementos (não os valores dos elementos diretamente). Embora o for...in funcione com arrays, ele não é ideal porque pode percorrer coisas que não são só os índices do array. Ele pode incluir propriedades extras que o array herdou e não garante que eles serão acessados em ordem. Para arrays, prefira usar for, for/of ou forEach.

For/of

Itera diretamente sobre os valores de objetos iteráveis. Esse loop é mais intuitivo e simples de usar quando você só precisa dos valores dos elementos, sem se preocupar com os índices.

Sintaxe:

for (let variável of objetoIterável) {
   // código a ser executado
}

Exemplo:

let frutas = ["maçã", "banana", "laranja"];

for (let fruta of frutas) {
   console.log(fruta);
} // saída: maçã
            banana
            laranja

Explicação:

  • fruta é uma variável que, a cada iteração, recebe o valor do próximo elemento do array frutas.
  • O for/of percorre automaticamente cada elemento no array, começando pelo primeiro e indo até o último.
  • A variável fruta recebe diretamente o valor de cada elemento em cada iteração.

While

O loop while executa um bloco de código enquanto uma condição especificada for verdadeira. Podemos utilizar a estrutura de repetição while caso seja necessário repetir um bloco de comandos por N vezes.

Diferente do for, o while executa o bloco de código enquanto uma condição é verdadeira. Ele é útil quando não sabemos exatamente o número de iterações necessárias ou quando a condição depende de outros fatores que podem mudar durante o loop.

Sintaxe:

while (condição) {
   // código a ser executado
}

A sintaxe do while, como vemos acima, é bem simples. Utilizamos a palavra reservada while, logo em seguida precisamos passar uma condição que será atendida enquanto ela retornar true.

Vamos usar o while para percorrer o array frutas como no exemplo anterior:

let frutas = ["maçã", "banana", "laranja"];
let i = 0; // Inicializamos a variável de contagem fora do loop

while (i < frutas.length) {
   console.log(frutas[i]);
   i++;
} // saída: maçã
            banana
            laranja

Explicação:

  • let i = 0;: Iniciamos i em 0 para que ele comece no primeiro índice do array.
  • while (i < frutas.length): O loop continuará enquanto i for menor que o comprimento do array (frutas.length).
  • console.log(frutas[i]);: Imprime o elemento atual no índice n.
  • i++;: Incrementa i em 1 a cada iteração, para que possamos acessar o próximo elemento do array.

Nota: Se você esquecer de aumentar a variável usada na condição usando i++, o loop nunca terminará. Isso irá travar o seu navegador.

Exemplo 2:

let i = 0;
while(i < 11) {
   console.log("5 x " + i + " = " + 5*i);
i++;
}

Neste caso, o código no loop será executado, repetidas vezes, desde que a variável i seja menor que 11. Ao atingir tal valor, o laço é quebrado. Executando o código acima teremos o seguinte resultado:

5 x 0 = 0
5 x 1 = 5
5 x 2 = 10
5 x 3 = 15
5 x 4 = 20
5 x 5 = 25
5 x 6 = 30
5 x 7 = 35
5 x 8 = 40
5 x 9 = 45
5 x 10 = 50

Do/while

O do/while tem o mesmo conceito que o while com uma única diferença, a condição é verificada após os comandos do bloco serem executados, ou seja, mesmo que a condição seja falsa, é garantia que o bloco será executado ao menos uma vez.

Sintaxe:

do {
    // Bloco de código a ser executado
} while (condição);

Podemos testar da seguinte forma:

let contador = 0;

do {
   console.log(contador);
   contador++;
} while (contador == 1);

Explicação:

1. Inicialização: A variável contador é definida como 0.

2. Primeira execução do bloco:

  • O bloco dentro do do será executado independentemente da condição porque ela é verificada apenas depois do bloco ser executado.
  • Ele executa console.log(contador);, imprimindo: "0"
  • Incrementa contador com contador++, agora contador = 1.

3. Verificação da condição:

  • A condição contador == 1 é avaliada.
  • Como contador é igual a 1, a condição é verdadeira, e o bloco será executado novamente.

4. Segunda execução do bloco:

  • O console.log é executado novamente, imprimindo: "1"
  • Incrementa contador com contador++, agora contador = 2.

5. Nova verificação da condição:

  • Agora a condição contador == 1 é falsa (porque contador = 2).
  • O loop termina.

O código gera a seguinte saída no console:

0
1

Nota: Não esqueça de incrementar a variável usada na condição, caso contrário o loop nunca vai acabar!

Métodos de controle de loops: break e continue

As palavras-chave break e continue são usadas em estruturas de repetição (for, while, do/while) e no switch para controlar o fluxo de execução. Vamos entender cada uma em detalhes:

Break

Finaliza o laço atual ou o bloco switch. Quando o JavaScript encontra a palavra-chave break, ele interrompe a execução do laço imediatamente e sai dele, mesmo que a condição do laço ainda seja verdadeira.

Exemplo: break com for

for (let i = 0; i <= 10; i++) {
   if (i === 5) {
      break;
   }
   console.log(i);
}

Quando i atinge o valor 5, o break é acionado e o loop termina imediatamente.

Saída:

0
1
2
3
4

Continue

Pula para a próxima iteração do laço, ignorando o código restante na iteração atual. Diferente do break, ele não encerra o laço, apenas salta para a próxima repetição.

Exemplo: continue com for

for (let i = 1; i <= 5; i++) {
   if (i === 3) {
      continue;
   }
   console.log(i);
}

Quando i é igual a 3, o continue ignora o resto do código (o console.log) e pula para a próxima iteração.

Saída:

1
2
4
5

Quando usar break e continue?

  • Use break quando você quiser parar completamente o laço ou o switch ao atingir uma condição específica.
  • Use continue quando você quiser ignorar parte do código de uma iteração e ir para a próxima repetição do laço.
