Manipulação do DOM

DOM (Document Object Model) é uma interface de programação (API) que representa a estrutura de um documento HTML, que pode ser modificada com uma linguagem de script como JavaScript. Em termos simples, ele transforma o seu HTML em um objeto JavaScript, permitindo que você manipule e interaja com os elementos da sua página web de forma dinâmica.

Nota: Interface, do latim inter (entre) + facies (faces), significa: ligação física ou lógica entre 2 sistemas ou partes que não poderiam ser conectados diretamente.

Este modelo utiliza uma linguagem neutra que, por meio de uma árvore hierárquica de diretórios, permite você acessar e interagir com qualquer elemento de seu documento com o uso do JavaScript.

Dessa forma, o JavaScript tem o poder de:

  • alterar, excluir e adicionar: elementos HTML, atributos HTML, eventos HTML, estilos CSS.

Ou seja, o DOM é o intermediador da comunicação da linguagem JavaScript com as tecnologias HTML5 e CSS3.

Ilustração de uma árvore DOM:

Parent (pai) - é quem está acima na árvore hierárquica.

Child (filho) - é quem está abaixo na árvore hierárquica.

É como se fosse uma árvore genealógica. Exemplos:

HEAD e BODY são filhos de HTML;

HTML é pai de HEAD e BODY;

HTML é filho de DOCUMENT.

Um dos grandes responsáveis por isso tudo é o objeto document que é responsável por conceder ao código Javascript todo o acesso a árvore DOM. Portanto, qualquer coisa criada pelo navegador Web no modelo da página Web poderá ser acessado através do objeto Javascript document.

É importante ressaltar que um documento HTML e sua representação DOM são a mesma coisa. A única diferença é que estes estão representados de formas diferentes. Ex: "uma garrafa" e "a bottle" são a mesma coisa, só que representadas em línguas diferentes. É a mesma ideia. Caso você altere o documento HTML, você também alterará o seu DOM e vice-versa.

Selecionando elementos

Para começar a manipular o DOM, você precisa primeiro selecionar os elementos que deseja modificar. Os métodos mais comuns para isso são:

document.getElementById() seleciona um elemento pelo seu ID.

Exemplo:

document.getElementById("myElement");

document.getElementsByName() seleciona elementos pelo NOME.

document.getElementsByTagName() seleciona elementos pela TAG.

document.getElementsByClassName() seleciona elementos pela CLASSE.

document.querySelector() seleciona o primeiro elemento que corresponda ao seletor CSS fornecido.

document.queySelectorAll() seleciona elementos pelo SELETOR CSS.
