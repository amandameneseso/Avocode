Manipulação do DOM

DOM (Document Object Model) é uma interface de programação (API) que representa a estrutura de um documento HTML, que pode ser modificada com uma linguagem de script como JavaScript. Em termos simples, ele transforma o seu HTML em um objeto JavaScript, permitindo que você manipule e interaja com os elementos da sua página web de forma dinâmica.

Nota: Interface, do latim inter (entre) + facies (faces), significa: ligação física ou lógica entre 2 sistemas ou partes que não poderiam ser conectados diretamente.

Este modelo utiliza uma linguagem neutra que, por meio de uma árvore hierárquica de diretórios, permite você acessar e interagir com qualquer elemento de seu documento com o uso do JavaScript.

Dessa forma, o JavaScript tem o poder de:

  • alterar, excluir e adicionar: elementos HTML, atributos HTML, eventos HTML, estilos CSS.

Ou seja, o DOM é o intermediador da comunicação da linguagem JavaScript com as tecnologias HTML5 e CSS3.

Ilustração de uma árvore DOM:

Parent (pai) - é quem está acima na árvore hierárquica.

Child (filho) - é quem está abaixo na árvore hierárquica.

É como se fosse uma árvore genealógica. Exemplos:

HEAD e BODY são filhos de HTML;

HTML é pai de HEAD e BODY;

HTML é filho de DOCUMENT.

Um dos grandes responsáveis por isso tudo é o objeto document que é responsável por conceder ao código Javascript todo o acesso a árvore DOM. Portanto, qualquer coisa criada pelo navegador Web no modelo da página Web poderá ser acessado através do objeto Javascript document.

É importante ressaltar que um documento HTML e sua representação DOM são a mesma coisa. A única diferença é que estes estão representados de formas diferentes. Ex: "uma garrafa" e "a bottle" são a mesma coisa, só que representadas em línguas diferentes. É a mesma ideia. Caso você altere o documento HTML, você também alterará o seu DOM e vice-versa.

Selecionando elementos

Para começar a manipular o DOM, você precisa primeiro selecionar os elementos que deseja modificar. Os métodos mais comuns para isso são:

document.getElementById() seleciona um elemento pelo seu ID.

Exemplo:

document.getElementById("myElement");

document.getElementsByName() seleciona elementos pelo NOME.

document.getElementsByTagName() seleciona elementos pela TAG.

document.getElementsByClassName() seleciona elementos pela CLASSE.

document.querySelector() seleciona o primeiro elemento que corresponda ao seletor CSS fornecido.

Nota: O Seletor de Classe é representado por: .; o Seletor de ID é representado por: #

document.queySelectorAll() seleciona elementos pelo SELETOR CSS.

querySelectorAll -> retorna uma NodeList com todos os elementos do seletor dentro dos parânteses.

Nota: NodeList é uma coleção de nós (nodes). Quando se tem uma estrutura baseada em árvore, o nó Node é o elemento básico da estrutura. No caso do JavaScript, o DOM é uma árvore de nós. A classe NodeList representa um conjunto desses nós, sendo retornado, por exemplo, pela função document.querySelectorAll.

Atente-se que alguns dos seletores estão no singular e outros no plural (element e elements). Caso utilize um seletor que está no plural (como document.getElementsByTagName ou document.getElementsByClassName), ele retornará uma coleção com todos os elementos selecionados (HTMLCollection).

Para especificar qual tag, name ou class você deseja acessar, você têm que usar, após os parênteses ( ), os colchetes [ ] e, dentro destes, você indicará qual a posição do elemento que você deseja acessar. Exemplo:

let elemento = document.getElementsByTagName('h1')[0];
console.log(elemento); // <h1>Manipulação do DOM<h1>

Neste exemplo selecionou-se, dentro dos parênteses, todas as tags <h1>. E, dentro dos colchetes, selecionou-se a tag <h1> que se encontra na posição 0, ou seja, a 1° tag <h1>.

Os seguintes elementos também podem ser acessados:

  • document.anchors
  • document.body
  • document.documentElement
  • document.embeds
  • document.forms
  • document.head
  • document.images
  • document.links
  • document.scripts
  • document.title

Manipulando elementos

Uma vez que você selecionou um elemento, você pode manipulá-lo de diversas formas. Para alterar o conteúdo de um elemento HTML, é possível usar a sintaxe:

<p id="p1">Olá, mundo!</p>

let texto = document.getElementById("p1");
texto.innerHTML = "Novo texto";
console.log(texto.innerHTML); // Novo texto

É possível simplificar:

let texto = document.getElementById("p1").innerHTML = "Novo texto";
console.log(texto.innerHTML); // Novo texto

Exemplo explicado:

  • O HTML acima contém um elemento <p> com id="p1";
  • Usamos o DOM para obter o elemento id="p1";
  • O método innerHTML altera o conteúdo desse elemento para "Novo texto".

Os métodos innerHTML, innerText e textContent são usados para acessar ou modificar o conteúdo de elementos HTML, mas eles possuem diferenças importantes em como manipulam o DOM e interpretam o conteúdo. Vamos detalhar cada um:

1. innerHTML: Retorna ou define o conteúdo HTML completo, incluindo tags e elementos filhos. Assim, permite inserir ou modificar HTML estruturado dentro de um elemento.

2. innerText: Retorna ou define o conteúdo de texto de um elemento, ignorando as tags HTML. Também ignora elementos ocultos por CSS (display: none ou visibility: hidden). Assim, permite trabalhar com o texto puro de um elemento, sem se preocupar com a formatação HTML.

3. textContent:Similar ao innerText, retorna ou define o conteúdo de texto de um elemento, mas inclui todo o texto, mesmo que esteja oculto.

Exemplo:

<div id="meuDiv">
   <h1>Título</h1>
   <p>Este é um parágrafo com <span style="display:none;">texto oculto</span>.</p>
</div>

let elemento = document.getElementById("meuDiv");

console.log(elemento.innerHTML);
// <h1>Título</h1>
   <p>Este é um parágrafo com <span style="display:none;">texto oculto</span>.</p>

console.log(elemento.innerText);
// Título
   Este é um parágrafo com .

console.log(elemento.textContent);
// Título
   Este é um parágrafo com texto oculto.
