Noções básicas de JavaScript

Declarações

Um programa de computador é uma lista de "instruções" a serem "executadas" por um computador. Em uma linguagem de programação, essas instruções são chamadas de declarações. Um programa JavaScript é uma lista de declarações de programação. As declarações são executadas, uma a uma, na mesma ordem em que são escritas.

Por exemplo, no JavaScript, uma declaração pode ser algo simples como declarar uma variável:

let mensagem = "Olá, Mundo!";

Ou executar uma operação mais complexa como a manipulação de elementos HTML:

document.getElementById("titulo").innerHTML = "Bem-vindo ao JavaScript!";

As declarações podem ser compostas por: Valores, Operadores, Expressões, Palavras-chave e Comentários.

Esta declaração diz ao navegador para escrever "JavaScript é legal" dentro de um elemento HTML com id="texto":

document.getElementById("texto").innerHTML = "JavaScript é legal";

Ponto e vírgula

Ponto e vírgula separam declarações. Adicione um ponto e vírgula no final de cada declaração executável. Exemplo:

let a, b, c; // Declara 3 variáveis
a = 5;       // Atribui o valor 5 para a
b = 6;       // Atribui o valor 6 para b
c = a + b;   // Atribui a soma de a e b para c

Quando separados por ponto e vírgula, várias declarações em uma linha são permitidas:

a = 5; b = 6; c = a + b;

Na web, você pode ver exemplos sem ponto e vírgula. Terminar declarações com ponto e vírgula não é obrigatório, mas altamente recomendado.

Espaço em branco

No JavaScript, os espaços em branco são ignorados pelo interpretador, exceto quando estão dentro de aspas (simples, duplas ou crases) ou quando fazem parte de uma string literal. Vamos entender isso em mais detalhes:

let nome = "Maria"; // Saída: Maria
let nome="Maria"; // Saída: Maria

As linhas acima são equivalente. O JavaScript não se importa com o número de espaços ou quebras de linha entre elementos do código. Isso significa que você pode formatar seu código com quantos espaços quiser para torná-lo mais legível.

Porém, dentro de aspas simples ('), aspas duplas (") ou crases (``), os espaços em branco são considerados parte da string e não são ignorados. Eles aparecem exatamente como foram escritos:

let texto = "Olá, mundo!";
console.log(texto); // Saída: Olá, mundo! (o espaço é mantido)

Bloco de código

As declarações podem ser agrupadas em blocos de código, dentro de chaves {...}. O objetivo dos blocos de código é definir declarações a serem executadas em conjunto. Um lugar onde você encontrará declarações agrupadas em blocos é em funções:

<button type="button" onclick="clicar()">Clique aqui</button>

<p id="texto1"></p>
<p id="texto2"></p>

<script>
function clicar() {
   document.getElementById("texto1").innerHTML = "Olá, Maria!";
   document.getElementById("texto2").innerHTML = "Seja bem-vindo(a)!";
}
</script>

O código acima cria um botão que, ao ser clicado, chama a função "clicar", que exibe textos dentro dos parágrafos. A primeira linha da função altera o conteúdo do elemento com id="texto1" para "Olá, Maria!". A segunda linha da função altera o conteúdo do elemento com id="texto2" para "Seja bem-vindo(a)!".

Palavras-chave

As declarações geralmente começam com uma palavra-chave para identificar a ação a ser executada. Palavras-chave são palavras reservadas. Palavras reservadas não podem ser usadas como nomes para variáveis. Aqui está uma lista de algumas das palavras-chave que você aprenderá neste tutorial:

Palavra-chave Descrição
var Declara uma variável
let Declara uma variável de bloco
const Declara uma constante de bloco
if Marca um bloco de instruções a serem executadas com uma condição
switch Marca um bloco de instruções a serem executadas em diferentes casos
for Marca um bloco de instruções a serem executadas em um loop
function Declara uma função
return Retorna um valor de uma função

Sintaxe

A sintaxe JavaScript é o conjunto de regras que define como escrever instruções corretamente para que o JavaScript seja compreendido pelo navegador. Essas regras especificam a estrutura que o código JavaScript deve seguir para que ele seja executado sem erros.

// Como criar variáveis:
var x;
let y;

// Como usar variáveis:
x = 5;
y = 6;
let z = x + y;

Principais elementos

1. Declarações: são comandos que o programa executa. Cada instrução termina com um ponto e vírgula (;). Exemplo:

let x = 5; // Declaração que define a variável 'x'
console.log(x); // Exibe o valor de 'x' no console

2. Variáveis: usadas para armazenar valores e podem ser declaradas usando as palavras-chave let, const ou var. Um sinal de igual (=) é usado para atribuir um valor a uma variável. Neste exemplo, 'idade' é definido como uma variável. Então, à 'idade' é atribuído (recebe) o valor 31:

let idade = 31;

3. Operadores: representam as operações matemáticas, relacionais e lógicas. Exemplo:

let soma = 5 + 3; // Operador de adição
let resultado = soma * 2; // Operador de multiplicação

4. Estruturas de controle: as estruturas de controle, como if, else, while, for, switch, são usadas para controlar o fluxo de execução do programa com base em condições ou repetições. Exemplo de "if":

let idade = 25;
if (idade >= 18) {
   console.log("Maior de idade");
} else {
   console.log("Menor de idade");
}

O código acima utiliza o if e else para verificar se uma pessoa é maior ou menor de idade com base no valor da variável idade. A variável idade é declarada e definida como 25. A estrutura if verifica se o valor de idade é maior ou igual a 18. Se for, a mensagem "Maior de idade" é exibida no console. Caso contrário, a mensagem "Menor de idade" é exibida.

5. Funções: as funções definem blocos de código que podem ser executados diversas vezes. O código abaixo é uma função que recebe um nome como argumento e retorna uma saudação personalizada:

function saudacao(nome) {
   return "Olá, " + nome + "!";
}
console.log(saudacao("Maria")); // Saída: Olá, Maria!

6. Arrays e objetos: arrays são listas ordenadas de valores e objetos são estruturas de dados que permitem armazenar valores de diferentes tipos.

Exemplo de array:

let frutas = ["maçã", "banana", "laranja"];

Exemplo de objeto:

let pessoa = {
   nome: "Maria",
   idade: 25,
   país: "Brasil"
};

7. Comentários: usados para adicionar notas ou instruções ao código. Eles são ignorados pelo interpretador do JavaScript. Código após barras duplas // ou entre /* e */ é tratado como um comentário. Também pode ser usado para impedir a execução do código, um uso adequado para testes. Exemplo:

// Este é um comentário
