Noções básicas de JavaScript

O que é JavaScript?

JavaScript é uma linguagem de script extremamente poderosa utilizada no lado do cliente. Ele é amplamente usado para melhorar a interação do usuário com páginas da web. Em outras palavras, o JavaScript permite tornar sua página da web mais dinâmica e interativa. Além disso, o JavaScript é amplamente aplicado no desenvolvimento de jogos e também no desenvolvimento de aplicativos móveis.

  • Linguagem de programação mais popular do mundo.
  • Linguagem de programação da Web.
  • Fácil de aprender.

História

O JavaScript foi desenvolvido por Brendan Eich em 1995, sendo inicialmente introduzido no navegador Netscape, que era muito popular na época. A linguagem foi criada para adicionar interatividade às páginas da web, algo que os navegadores ainda não suportavam de forma nativa. Originalmente, o JavaScript era chamado de LiveScript, mas foi renomeado para JavaScript como uma estratégia de marketing para aproveitar a popularidade da linguagem Java, que estava em alta naquele momento.

A sintaxe do JavaScript foi influenciada principalmente pela linguagem C, o que torna sua curva de aprendizado mais amigável para quem já possui experiência com linguagens baseadas em C, como C++ e Java. Além disso, algumas de suas características foram inspiradas por outras linguagens, como o Scheme e o Self, especialmente no que diz respeito à sua natureza funcional e ao modelo de prototipagem.

Com o tempo, o JavaScript evoluiu significativamente. Hoje, ele é amplamente utilizado tanto no front-end (navegadores) quanto no back-end (com o Node.js), além de ser uma escolha popular para o desenvolvimento de aplicativos móveis, jogos e até mesmo inteligência artificial. Essa versatilidade fez do JavaScript uma das linguagens mais populares e essenciais para qualquer desenvolvedor web moderno.

Como executar JavaScript?

Sendo uma linguagem de script, o JavaScript não é executado de forma independente, como linguagens de programação tradicionais. Ele depende de um ambiente de execução, e nos navegadores modernos, esse ambiente é o motor JavaScript integrado. Por exemplo, o Google Chrome utiliza o motor V8, enquanto o Firefox usa o SpiderMonkey. Esses motores são responsáveis por interpretar e executar o código JavaScript, permitindo que ele interaja com os elementos da página e ofereça funcionalidades dinâmicas.

Quando um usuário acessa uma página HTML que contém JavaScript, o navegador faz o download do código junto com o restante da página. A execução ocorre diretamente no cliente, ou seja, no computador do usuário, garantindo uma experiência interativa sem a necessidade de comunicação constante com o servidor.

Onde mais o JavaScript pode ser executado?

Embora o navegador seja o ambiente mais comum para a execução de JavaScript, ele não se limita a isso. Com a evolução da linguagem, surgiram outras plataformas que permitem executar JavaScript fora do navegador, como:

  • Node.js: Uma plataforma baseada no motor V8 do Chrome, usada para executar JavaScript no lado do servidor. Com o Node.js, é possível criar APIs, servidores web e até mesmo aplicativos de linha de comando.
  • Motores embutidos em outras aplicações: Algumas ferramentas e softwares incluem motores JavaScript para permitir scripts customizados, como o Adobe Acrobat e o Photoshop.
  • Desenvolvimento de jogos e aplicativos móveis: Frameworks como React Native e Electron permitem que JavaScript seja usado para criar aplicativos móveis e desktop.

O que o JavaScript pode fazer?

1. Mudar o conteúdo HTML

Um dos muitos métodos JavaScript é o getElementById(). O exemplo abaixo "encontra" um elemento HTML (com id="demo"), e altera o conteúdo do elemento (innerHTML) para "Olá JavaScript":

            
        <!DOCTYPE html>
        <html>
        <body>

        <p id="demo">Mudar o conteúdo HTML.</p>

        <button type="button" onclick='document.getElementById("demo").innerHTML = "Olá, JavaScript!"'>Clique aqui</button>

        </body>
        </html>

Observe no código anterior que, ao clicar no botão definido como "Clique aqui", o texto com id="demo" mudará para o texto definido dentro do método "onclick" (Olá, JavaScript!):

        <p id="demo">Olá, JavaScript!</p>

Nota: JavaScript aceita aspas duplas e simples.

2. Alterar valores de atributos no HTML

Neste exemplo, o JavaScript altera o valor do atributo src (fonte) de uma tag <img>:

            
        <!DOCTYPE html>
        <html>
        <body>

        <img src="imagens/pic_bulboff.webp" id="luz" style="display: flex; margin: auto;">

        <button onclick="document.getElementById('luz').src='pic_bulbon.webp'">Ligar a luz</button>

        <button onclick="document.getElementById('luz').src='pic_bulboff.webp'">Apagar a luz</button>

        </body>
        </html>

3. Alterar estilos no CSS

Alterar um estilo de um elemento HTML é semelhante a alterar um atributo no HTML:

            
        <!DOCTYPE html>
        <html>
        <body>

        <p id="demo">Alterar estilos no CSS</p>

        <button type="button" onclick="document.getElementById('demo').style.fontSize='35px'">Clique aqui</button>

        </body>
        </html>

Note que ao clicar no botão "Clique aqui", o tamanho da fonte do parágrafo com id="demo" será alterado.

Como adicionar e executar JavaScript em uma página HTML?

Você pode incluir JavaScript em uma página HTML de várias formas:

1. Script embutido (inline)

O código JavaScript é inserido diretamente dentro de uma tag <script> no body do arquivo HTML:

            
        <script>
            alert("Olá, mundo!");
        </script>

2. Arquivo externo

O código JavaScript é armazenado em um arquivo separado com extensão .js, e o arquivo é vinculado à página HTML. Para usar um script externo, coloque o nome do arquivo de script no src (fonte) dentro da tag <script>:

        <script src="script.js"></script>

3. Eventos

Também é possível incluir JavaScript diretamente nos atributos de elementos HTML, como onclick, onmouseover etc.:

        <button onclick="alert('Você clicou no botão!')">Clique aqui</button>
