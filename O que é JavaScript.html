O que é JavaScript?

JavaScript é uma linguagem de script extremamente poderosa utilizada no lado do cliente. Ele é amplamente usado para melhorar a interação do usuário com páginas da web. Em outras palavras, o JavaScript permite tornar sua página da web mais dinâmica e interativa. Além disso, o JavaScript é amplamente aplicado no desenvolvimento de jogos e também no desenvolvimento de aplicativos móveis.

  • Linguagem de programação mais popular do mundo.
  • Linguagem de programação da Web.
  • Fácil de aprender.

História

O JavaScript foi desenvolvido por Brendan Eich em 1995, sendo inicialmente introduzido no navegador Netscape, que era muito popular na época. A linguagem foi criada para adicionar interatividade às páginas da web, algo que os navegadores ainda não suportavam de forma nativa. Originalmente, o JavaScript era chamado de LiveScript, mas foi renomeado para JavaScript como uma estratégia de marketing para aproveitar a popularidade da linguagem Java, que estava em alta naquele momento.

A sintaxe do JavaScript foi influenciada principalmente pela linguagem C, o que torna sua curva de aprendizado mais amigável para quem já possui experiência com linguagens baseadas em C, como C++ e Java. Além disso, algumas de suas características foram inspiradas por outras linguagens, como o Scheme e o Self, especialmente no que diz respeito à sua natureza funcional e ao modelo de prototipagem.

Com o tempo, o JavaScript evoluiu significativamente. Hoje, ele é amplamente utilizado tanto no front-end (navegadores) quanto no back-end (com o Node.js), além de ser uma escolha popular para o desenvolvimento de aplicativos móveis, jogos e até mesmo inteligência artificial. Essa versatilidade fez do JavaScript uma das linguagens mais populares e essenciais para qualquer desenvolvedor web moderno.

Como executar JavaScript?

Sendo uma linguagem de script, o JavaScript não é executado de forma independente, como linguagens de programação tradicionais. Ele depende de um ambiente de execução, e nos navegadores modernos, esse ambiente é o motor JavaScript integrado. Por exemplo, o Google Chrome utiliza o motor V8, enquanto o Firefox usa o SpiderMonkey. Esses motores são responsáveis por interpretar e executar o código JavaScript, permitindo que ele interaja com os elementos da página e ofereça funcionalidades dinâmicas.

Quando um usuário acessa uma página HTML que contém JavaScript, o navegador faz o download do código junto com o restante da página. A execução ocorre diretamente no cliente, ou seja, no computador do usuário, garantindo uma experiência interativa sem a necessidade de comunicação constante com o servidor.

Onde mais o JavaScript pode ser executado?

Embora o navegador seja o ambiente mais comum para a execução de JavaScript, ele não se limita a isso. Com a evolução da linguagem, surgiram outras plataformas que permitem executar JavaScript fora do navegador, como:

  • Node.js: Uma plataforma baseada no motor V8 do Chrome, usada para executar JavaScript no lado do servidor. Com o Node.js, é possível criar APIs, servidores web e até mesmo aplicativos de linha de comando.
  • Motores embutidos em outras aplicações: Algumas ferramentas e softwares incluem motores JavaScript para permitir scripts customizados, como o Adobe Acrobat e o Photoshop.
  • Desenvolvimento de jogos e aplicativos móveis: Frameworks como React Native e Electron permitem que JavaScript seja usado para criar aplicativos móveis e desktop.

O que o JavaScript pode fazer?

1. Mudar o conteúdo HTML

Um dos muitos métodos JavaScript é o getElementById(). O exemplo abaixo "encontra" um elemento HTML (com id="demo"), e altera o conteúdo do elemento (innerHTML) para "Olá JavaScript":

<!DOCTYPE html>
<html>
<body>

<p id="demo">Mudar o conteúdo HTML.</p>

<button type="button" onclick='document.getElementById("demo").innerHTML = "Olá, JavaScript!"'>Clique aqui</button>

</body>
</html>

Observe no código anterior que, ao clicar no botão definido como "Clique aqui", o texto com id="demo" mudará para o texto definido dentro do método "onclick" (Olá, JavaScript!):

<p id="demo">Olá, JavaScript!</p>

Nota: JavaScript aceita aspas duplas e simples.

2. Alterar valores de atributos no HTML

Neste exemplo, o JavaScript altera o valor do atributo src (fonte) de uma tag <img>:

<!DOCTYPE html>
<html>
<body>
   <img src="imagens/pic_bulboff.webp" id="luz" style="display: flex; margin: auto;">
   <button onclick="document.getElementById('luz').src='pic_bulbon.webp'">Ligar a luz</button>
   <button onclick="document.getElementById('luz').src='pic_bulboff.webp'">Apagar a luz</button>
</body>
</html>

3. Alterar estilos no CSS

Alterar um estilo de um elemento HTML é semelhante a alterar um atributo no HTML:

<!DOCTYPE html>
<html>
<body>
   <p id="demo">Alterar estilos no CSS</p>
   <button type="button" onclick="document.getElementById('demo').style.fontSize='35px'">Clique aqui</button>
</body>
</html>

Note que ao clicar no botão "Clique aqui", o tamanho da fonte do parágrafo com id="demo" será alterado.

Como adicionar e executar JavaScript em uma página HTML?

Você pode incluir JavaScript em uma página HTML de várias formas:

1. Script embutido (inline)

O código JavaScript é inserido diretamente dentro de uma tag <script> no body do arquivo HTML:

<script>
   alert("Olá, mundo!");
</script>

2. Arquivo externo

O código JavaScript é armazenado em um arquivo separado com extensão .js, e o arquivo é vinculado à página HTML. Para começar, vamos examinar como adicionar JavaScript à sua página:

1. Vá para o seu site de teste e crie uma nova pasta chamada scripts. Dentro da pasta "scripts", crie um novo documento de texto chamado main.js e salve-o.

2. Em seu arquivo "index.html", insira este código em uma nova linha, logo antes da tag de fechamento </body>:

<script src="scripts/main.js"></script>

3. Isso faz o mesmo trabalho que o elemento <link> para CSS. Ele aplica o JavaScript à página, para que possa afetar o HTML.

Dica: Coloque o <script> no final do <body>. Isso garante que todos os elementos da página já estejam carregados.

Colocar scripts em arquivos externos tem algumas vantagens:

  • Um arquivo .js externo pode ser reutilizado em várias páginas do site, reduzindo a duplicação de código.
  • Mantém o HTML mais limpo e separado do JavaScript, facilitando a leitura e manutenção do código.
  • Atualizar o script em um único arquivo externo aplica as mudanças automaticamente em todas as páginas que o utilizam. Isso economiza tempo e esforço.
  • Arquivos JavaScript externos podem ser armazenados no cache do navegador, o que reduz o tempo de carregamento em visitas subsequentes.

3. Eventos

Também é possível incluir JavaScript diretamente nos atributos de elementos HTML, como onclick, onmouseover etc.:

<button onclick="alert('Você clicou no botão!')">Clique aqui</button>

Possibilidades de Exibição do JavaScript

O JavaScript pode exibir dados de várias formas para o usuário ou desenvolvedor. Abaixo estão as principais possibilidades, com explicações e exemplos:

1. innerHTML

Permite inserir ou alterar o conteúdo de elementos HTML diretamente. Primeiramente, para acessar um elemento HTML, o JavaScript pode usar o método document.getElementById. O conteúdo dentro do id pode ser alterado utilizando innerHTML. Exemplo:

<p id="mensagem">Texto inicial.</p>
<script>
   document.getElementById("mensagem").innerHTML = "Texto alterado com JavaScript!";
</script>

Alterando o texto com id="mensagem" através do innerHTML, o novo texto dentro do parágrafo será "Texto alterado com JavaScript!"

2. document.write()

Insere texto ou HTML diretamente na página durante o carregamento. Usar document.write() depois que um documento HTML é carregado, irá excluir todo o HTML existente. O document.write() é útil em testes e aprendizado inicial porque é simples e direto. No entanto, para projetos reais ou avançados, métodos modernos como innerHTML são mais adequados. Exemplo:

<script>
   document.write(5 + 6);
</script>

3. window.alert()

Exibe uma mensagem em uma caixa de alerta. Bom para mensagens de notificação ou testes simples. Pode ser intrusivo para os usuários. Exemplo:

<script>
   window.alert("Olá! Esta é uma mensagem de alerta.");
</script>

Nota: Você pode pular a palavra-chave window. Em JavaScript, o objeto window é um objeto de escopo global. Isso significa que variáveis, propriedades e métodos por padrão pertencem ao objeto window. Isso também significa que especificar a palavra-chave window é opcional.

4. console.log()

Exibe mensagens no console do navegador, geralmente usado para depuração e testes. Não interfere na interface do usuário e é útil para desenvolvedores. Para acessar o console, use a ferramenta "Inspecionar" no navegador (geralmente com F12) e acesse a aba "Console".

<script>
   console.log("Olá, JavaScript!");
</script>

Resumindo:

  • innerHTML manipula o conteúdo da página dinamicamente.
  • document.write insere texto ou HTML diretamente na página durante o carregamento, excluindo o HTML preeexistente.
  • alert() exibe mensagem em uma caixa de alerta.
  • console.log() exibe mensagem no console do navegador. Preferível durante o desenvolvimento para rastrear erros ou depurar scripts.

Características do JavaScript

Tipagem mutável

Uma linguagem com tipagem mutável (também chamada de tipagem dinâmica) é aquela em que o tipo de uma variável pode mudar durante a execução do programa. Isso significa que a variável não tem um tipo fixo e pode armazenar diferentes tipos de dados ao longo do tempo.

Por exemplo, em linguagens com tipagem mutável, você pode fazer algo assim:

let valor = 42; // Inicialmente, 'valor' é do tipo Number
console.log(typeof valor); // Saída: "number"

valor = "Agora sou uma string"; // Agora, 'valor' é do tipo String
console.log(typeof valor); // Saída: "string"

Em linguagens com tipagem mutável, você não precisa declarar explicitamente o tipo das variáveis. As variáveis podem ser reutilizadas para armazenar diferentes tipos de dados. Embora a tipagem mutável torne a linguagem mais flexível, a mudança de tipos pode levar a bugs difíceis de identificar, como operações incompatíveis entre tipos.

Case-sensitive

Uma linguagem ser case-sensitive significa que ela diferencia letras maiúsculas de minúsculas ao interpretar identificadores como nomes de variáveis, funções, classes ou outros elementos do código. Ou seja, variavel, Variavel e VARIAVEL são considerados identificadores diferentes. Por exemplo:

let nome = "João";
let Nome = "Maria";

console.log(nome); // Saída: "João"
console.log(Nome); // Saída: "Maria"

No exemplo acima, nome e Nome são duas variáveis diferentes devido à diferenciação de maiúsculas e minúsculas.

Alto nível

Linguagem de alto nível define uma linguagem projetada para ser mais próxima da linguagem humana e mais abstrata em relação ao funcionamento interno do computador. Em outras palavras, JavaScript abstrai muitos detalhes técnicos, como a manipulação direta da memória ou o controle do hardware, permitindo que o programador se concentre mais na lógica do programa e menos nos aspectos de baixo nível do sistema.

Linguagem Full Stack

Uma linguagem Full Stack pode ser usada tanto no front-end (lado cliente) quanto no back-end (lado servidor) de uma aplicação. Essa versatilidade permite aos desenvolvedores criar sistemas completos usando uma única linguagem, o que reduz a complexidade do desenvolvimento e melhora a consistência entre as partes do projeto.

JavaScript foi originalmente criado para o front-end, sendo executado nos navegadores para criar interfaces interativas. Com o tempo, sua capacidade foi ampliada para o back-end, especialmente com o advento do Node.js, que trouxe a execução do JavaScript para o lado servidor.
