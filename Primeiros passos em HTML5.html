O que é HTML?

HTML, ou Hyper Text Markup Language, foi criado por Tim Berners-Lee, Robert Cailliau e outros, a partir de 1989. Essa linguagem é usada para estruturar páginas na web. O termo "hipertexto" refere-se à capacidade de incluir links no documento, permitindo ao leitor navegar para outras partes do mesmo documento ou para páginas completamente diferentes. A versão mais recente, conhecida como HTML5, trouxe diversas melhorias e novos recursos.

Uma linguagem de marcação é uma maneira na qual os computadores falam uns com os outros para controlar como o texto é processado e apresentado. Para fazer isso, o HTML usa duas coisas: tags e atributos.

Ou seja, HTML é a linguagem principal usada para criar a estrutura da maioria dos sites. Ele serve para construir páginas e torná-las funcionais. Já o visual das páginas, como cores, fontes e layout, é tratado com CSS — algo que exploraremos em um tutorial futuro. Por enquanto, vamos nos concentrar em aprender a como construir antes de projetar.

  • HTML significa Hyper Text Markup Language;
  • É a linguagem de marcação padrão para criar páginas da Web;
  • Descreve a estrutura de uma página da Web;
  • Consiste em uma série de elementos;
  • Os elementos HTML informam ao navegador como exibir o conteúdo;
  • Elementos HTML rotulam partes do conteúdo como "este é um título", "isso é um parágrafo", "este é um link", etc.

Uma Linguagem de Marcação é uma maneira pela qual os computadores falam entre si para controlar como o texto é processado e apresentado. Um navegador não exibe as tags HTML, mas as usa para determinar como exibir o documento. Para fazer isso, o HTML usa duas coisas: tags e atributos.

O que é um elemento HTML?

Um elemento HTML é definido por uma tag inicial, algum conteúdo e uma tag final:

<tagname> O conteúdo vai aqui... </tagname>

Ou seja, elemento é tudo, desde a tag inicial até a tag final:

<h1> Meu primeiro título </h1>

<p> Meu primeiro parágrafo </p>

Nota: Alguns elementos HTML não têm conteúdo (como o <br>). Esses elementos são chamados de elementos vazios. Elementos vazios não têm uma etiqueta final!

O que é uma tag?

Tags são usadas para marcar o início de um elemento HTML e geralmente são colocadas entre parênteses angulares. Um exemplo de tag é: <h1>.

A maioria das tags devem ser abertas (<p>) e fechadas (</p>) para funcionarem.

O que são atributos?

Atributos contêm informações adicionais. Atributos são colocados em uma tag de abertura e informações adicionais são colocadas dentro.

Um exemplo de atributo é:

<img src="mydog.jpg" alt="A photo of my dog.">

Neste caso, a fonte da imagem (src) e o texto alternativo (alt) são atributos da tag <img>.

Um documento HTML simples

<!DOCTYPE html>
<html>
<head>
   <title>Nome da página</title>
</head>
<body>

   <h1>Meu primeiro título</h1>
   <p>Meu primeiro parágrafo</p>

</body>
</html>

Explicando o exemplo:

A declaração <!DOCTYPE html> define que este documento é um documento HTML5;

O <html> é o elemento raiz de uma página HTML. Esta tag sinaliza que daqui em diante vamos escrever em código HTML;

O <head> contém todos os metadados da página — coisas destinadas principalmente a mecanismos de busca e outros programas de computador;

O <title> especifica um título para a página (que é mostrada na barra de título do navegador ou na guia da página);

O <body> define o corpo do documento, e é um recipiente para todo o conteúdo visível;

O <h1> define um título grande (de nível 1).

O <p> define um parágrafo.

É assim que sua página HTML é estruturada visualmente:

<html>
<head>
<title> Nome da página </title>
</head>
<body>
<h1> Meu primeiro título </h1>
<p> Meu primeiro parágrafo </p>
</body>
</html>

Nota: O conteúdo dentro da secção <body> será exibido em um navegador. O conteúdo dentro do elemento <title> será mostrado na barra de título do navegador ou na guia da página.

Escrevendo o primeiro código

Para iniciar, utilizaremos o editor Visual Studio Code para escrever os códigos do arquivo HTML5. O arquivo deve ter o formato ".html". Para criar o código base, vá até as linhas do arquivo criado e digite apenas um ponto de exclamação "!". O código será criado automaticamente e precisa apenas de algumas pequenas atualizações:

linha 2: <html lang="pt-br">

linha 6: Alterar o nome do título. Este será o título que aparecerá na barra do navegador.

E clique em salvar. Salvaremos esse arquivo como “index.html” em uma nova pasta chamada “minha página”.

    Momento da verdade: Clique no arquivo recém-salvo e sua primeira página da web deve abrir no seu navegador padrão. Pode não ser bonito, mas é seu... todo seu. *Risada maligna*

Bem, não vamos nos empolgar; ainda temos muitos recursos excelentes que podemos adicionar à sua página!

Adicionando texto ao HTML

Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo no meio do par de tags <p> e </p>.

Títulos em HTML são conhecidos como headings. Os títulos possuem seis níveis de hierarquia. Para isso, usamos as tags <h1>, <h2>, <h3>, <h4>, <h5> e <h6>. Como você deve ter adivinhado, <h1> e <h2> devem ser usadas para os títulos mais importantes, enquanto as tags restantes devem ser usadas para subtítulos e textos menos importantes.

Nota: Os mecanismos de busca usam essa ordem para decifrar quais informações são mais importantes em uma página.

Se precisar quebrar o texto em algum lugar específico
como esse, você pode usar a tag <br>.

Já a tag <hr> faz aparecer uma linha horizontal. Abaixo vemos um exemplo de código utilizando essas tags:

<!DOCTYPE html>
<html lang="pt-br">
<head>
   <meta charset="UTF-8">
   <meta name="viewport" content="width=device-width, initial-scale=1.0">
   <title>Minha primeira página</title>
</head>
<body>
   <h1>Olá, Mundo!</h1>
   <hr>
   <p>Esse é o meu primeiro documento HTML! Estou muito feliz!</p>
   <p>Este é um momento único! <br>
   Estou criando um site!</p>
</body>
</html>
