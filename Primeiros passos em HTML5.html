O que é HTML?

HTML, ou Hyper Text Markup Language, foi criado por Tim Berners-Lee, Robert Cailliau e outros, a partir de 1989. Essa linguagem é usada para estruturar páginas na web. O termo "hipertexto" refere-se à capacidade de incluir links no documento, permitindo ao leitor navegar para outras partes do mesmo documento ou para páginas completamente diferentes. A versão mais recente, conhecida como HTML5, trouxe diversas melhorias e novos recursos.

Uma linguagem de marcação é uma maneira na qual os computadores falam uns com os outros para controlar como o texto é processado e apresentado. Para fazer isso, o HTML usa duas coisas: tags e atributos.

Ou seja, HTML é a linguagem principal usada para criar a estrutura da maioria dos sites. Ele serve para construir páginas e torná-las funcionais. Já o visual das páginas, como cores, fontes e layout, é tratado com CSS — algo que exploraremos em um tutorial futuro. Por enquanto, vamos nos concentrar em aprender a como construir antes de projetar.

Escrevendo o primeiro código

Para iniciar, utilizaremos o editor Visual Studio Code para escrever os códigos do arquivo HTML5. O arquivo deve ter o formato ".html". Para criar o código base, vá até as linhas do arquivo criado e digite apenas um ponto de exclamação "!". O código será criado automaticamente e precisa apenas de algumas pequenas atualizações:

linha 2: <html lang="pt-br">

linha 6: Alterar o nome do título. Este será o título que aparecerá na barra do navegador.

Entendendo parágrafos e títulos

Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo no meio do par de tags <p> e </p>.

Títulos em HTML são conhecidos como headings. Os títulos possuem seis níveis de hierarquia. Para isso, usamos as tags <h1>, <h2>, <h3>, <h4>, <h5> e <h6>.

Se precisar quebrar o texto em algum lugar específico
como esse, você pode usar a tag <br>.

Já a tag <hr> faz aparecer uma linha horizontal. Abaixo vemos um exemplo de código utilizando essas tags:

<!DOCTYPE html>
<html lang="pt-br">
<head>
   <meta charset="UTF-8">
   <meta name="viewport" content="width=device-width, initial-scale=1.0">
   <title>Minha primeira página</title>
</head>
<body>
   <h1>Olá, Mundo!</h1>
   <hr>
   <p>Esse é o meu primeiro documento HTML! Estou muito feliz!</p>
   <p>Este é um momento único! <br>
   Estou criando um site!</p>
</body>
</html>
