Principais tags e suas funções

Estrutura básica

  • <!DOCTYPE html>: Declara que o documento é HTML5.
  • <html>: Elemento raiz do documento HTML.
  • <head>: Contém metadados sobre o documento.
  • <title>: Define o título do documento, exibido na aba do navegador.
  • <meta>: Fornece metadados, como charset e descrição.
  • <link>: Vincula folhas de estilo externas.
  • <style>: Contém CSS interno.
  • <script>: Inclui scripts JavaScript.

Estrutura do corpo

  • <body>: Contém o conteúdo visível da página.
  • <header> Define um cabeçalho para uma seção ou o documento.
  • <nav>: Define uma seção de navegação.
  • <main>: Define o conteúdo principal da página.
  • <section>: Define uma seção temática no documento.
  • <article>: Representa um conteúdo independente e autocontido.
  • <aside>: Contém conteúdo relacionado, como uma barra lateral.
  • <footer>: Define um rodapé para uma seção ou o documento.

Texto e estilo

  • <h1> a <h6>: Definem cabeçalhos de diferentes níveis.
  • <p>: Define um parágrafo.
  • <a>: Define um hiperlink.
  • <img>: Insere uma imagem.
  • <div>: Um contêiner genérico para agrupar conteúdo.
  • <span>: Um contêiner em linha para aplicar estilos.
  • <strong>: Indica ênfase forte; normalmente exibe em negrito.
  • <em>: Indica ênfase; normalmente exibe em itálico.
  • <small>: Define texto de menor importância.
  • <cite>: Indica uma referência a uma fonte.
  • <q>: Define uma citação em linha.
  • <blockquote>: Define uma citação em bloco.
  • <abbr>: Define uma abreviação.
  • <time>: Representa uma data ou hora.
  • <mark>: Destaca texto importante.
  • <del>: Indica texto que foi removido.
  • <ins>: Indica texto que foi inserido.
  • <sub>: Define texto como subscrito.
  • <sup>: Define texto como sobrescrito.

Listas

  • <ul>: Define uma lista não ordenada.
  • <ol>: Define uma lista ordenada.
  • <li>: Define um item em uma lista.
  • <dl>: Define uma lista de definições.
  • <dt>: Define um termo em uma lista de definições.
  • <dd>: Define a descrição de um termo em uma lista de definições.

Tabelas

  • <table>: Define uma tabela.
  • <tr>: Define uma linha na tabela.
  • <th>: Define uma célula de cabeçalho na tabela.
  • <td>: Define uma célula de dados na tabela.
  • <caption>: Fornece um título para a tabela.
  • <thead>: Agrupa o cabeçalho da tabela.
  • <tbody>: Agrupa o corpo da tabela.
  • <tfoot>: Agrupa o rodapé da tabela.

Formulários

  • <form>: Define um formulário para coleta de dados.
  • <input>: Define um campo de entrada.
  • <textarea>: Define uma área de texto.
  • <button>: Define um botão.
  • <select>: Cria um menu suspenso.
  • <option>: Define uma opção em um menu suspenso.
  • <optgroup>: Agrupa opções dentro de um menu suspenso.
  • <label>: Define um rótulo para um elemento de formulário.
  • <fieldset>: Agrupa elementos relacionados em um formulário.
  • <legend>: Define um título para um <fieldset>.
  • <datalist>: Fornece uma lista de opções para um campo de entrada.
  • <output>: Representa o resultado de um cálculo.

Multimídia

  • <audio>: Insere um áudio na página.
  • <video>: Insere um vídeo na página.
  • <source>: Especifica múltiplas fontes para um elemento de mídia.
  • <track>: Adiciona faixas de texto a vídeos (como legendas).
  • <img>: Insere uma imagem.
  • <canvas>: Permite desenhar gráficos e imagens com JavaScript.
  • <svg>: Insere gráficos vetoriais escaláveis.

Semântica e acessibilidade

  • <address>: Contém informações de contato.
  • <bdi>: Isola uma parte do texto para evitar que o navegador altere a direção.
  • <bdo>: Controla a direção do texto.
  • <progress>: Representa o progresso de uma tarefa.
  • <meter>: Representa uma medição dentro de um intervalo.

Scripts e APIs

  • <script>: Inclui scripts JavaScript.
  • <noscript>: Define conteúdo alternativo para navegadores que não suportam scripts.
  • <template>: Define conteúdo que pode ser clonado e reutilizado.
  • <slot>: Define um espaço reservado em um componente web.

Elementos de aplicação

  • <details>: Cria um controle que o usuário pode abrir ou fechar.
  • <summary>: Define um título para um elemento <details>.
  • <dialog>: Representa uma caixa de diálogo ou interface de interação.

Outros

  • <meta charset="UTF-8">: Define a codificação de caracteres do documento.
  • <link rel="icon">: Define um ícone para a página.
