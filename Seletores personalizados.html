Seletores personalizados

Agora que nós já passamos pelos elementos mais importantes do design, que são as cores, as imagens e as fontes, vamos falar um pouco sobre formatações especiais que vão permitir organizar esses conteúdos na tela, indicando seus tamanhos, espaçamentos, sombras e tudo mais.

"id" e "class"

No CSS, sempre que fazemos uma configuração de estilo para um elemento do HTML em um seletor, como por exemplo um h1, todas as unidades desse elemento ficam igualmente marcadas com essa configuração de estilo.

No entanto, às vezes, de forma específica, iremos precisar personalizar apenas uma unidade de um elemento, sem alterar o estilo das demais unidades desse mesmo elemento (uma unidade de h1, por exemplo). Para fazer isso, temos que aprender a utilizar os comandos de id (#) e de class (.) de maneira eficiente.

Ao criar nosso documento em HTML, podemos identificar um determinado elemento único com um "id" ou agrupar elementos múltiplos que devam ter características semelhantes com um "class".

Para usar o "id" dentro do código HTML, devemos dar uma identificação para a tag específica do HTML que queremos personalizar, essa identificação é feita dentro da tag desejada com o comando "id". Essa identificação irá diferenciar o h1 com "id" dos outros h1s sem "id". Exemplo:


        <h1 id="principal">Criando Sites com HTML e CSS</h1>

Após inserir o comando "id" dentro da tag, é preciso dar um nome (de sua preferência) para essa identificação. Feita a configuração, isso vai diferenciar essa unidade (com id) de todos os seus pares (mesmo elemento). Com isso iremos conseguir personalizar configurações de estilo para determinadas unidades específicas, sem alterar os seus pares.

Agora, para configurar especificamente o estilo desse id que denominamos como "principal" no exemplo, devemos ir até o nosso código CSS e abrir um seletor personalizado para esse id que criamos. Esse seletor deve possuir o nome do elemento a que a unidade marcada pertence e o identificador "#principal" (# + nome do id), da seguinte forma:

          h1#principal {font-size: 20px;

        }

Dessa forma, toda configuração que fizermos dentro desse seletor personalizado será válida somente para a tag que tem o respectivo id.

Observação: Não precisamos necessariamente escrever o nome do elemento a que a unidade marcada pertence (h1, no caso do exemplo). Isso faria com que conseguíssemos colocar qualquer tag que quiséssemos como esse id "#principal", podendo alterar o estilo de várias unidades de elementos de forma personalizada. Mas é recomendado escrever "elemento + # + nome do id", com o nome do elemento na frente, já que só é permitido usar somente um id e o seu respectivo nome por documento HTML. O uso de "class" que é o indicado para a alteração personalizada do estilo de várias unidades.

Quanto aos ids, nós temos uma grande limitação: dentro de um documento HTML, só se pode ter um elemento com o "id" e o seu respectivo nome. Você pode ter mais de um id em sua página tranquilamente. O que não pode é utilizar dois com o mesmo nome. O id é como um CPF (1 número por pessoa) enquanto a class é o nome (várias pessoas podem ter o mesmo nome).

Pensando nisso, quando temos duas unidades de elementos ou mais e queremos aplicar uma configuração personalizada de estilo em todas elas, nós não devemos usar o "id", e sim o "class".

Uma "class" também faz configurações personalizadas, mas podemos ter várias unidades de elemento com uma mesma classe. Sendo que, diferente do id, que é identificado com "#" no CSS, o class é identificado com "." no CSS.

No HTML:


        <h1 class="principal">Título principal</h1>


        <p class="principal">Lorem ipsum</h1>

No CSS:


        .principal {font-size: 20px;}

Ao criar o seletor personalizado de uma classe no CSS, geralmente não especificamos um nome de elemento na frente da sua representação ( . ) como é feito no id (ex: h1#teste), simplesmente inserimos o ponto e o nome da class. Dessa forma, toda unidade de elemento que pertence à determinada classe vai simplesmente incorporar as configurações de estilo do respectivo seletor personalizado sem depender de uma especificação de p, h1, h2 e etc.

Observação: No HTML temos a tag "<span>", ela não tem uma função específica, serve apenas para demarcar um trecho de texto caso quisermos fazer algum tipo de alteração de estilo nesse trecho com uma class. Ele deve ser usado somente quando nenhum outro elemento semântico for apropriado. "<span>" é muito parecido com o elemento "<div>", entretanto "<div>" é um elemento de nível de bloco enquanto "<span>" é um elemento em linha. Exemplo:


        <span class="negrito">Lorem ipsum</span>

Referente às classes, uma coisa que podemos fazer é colocar duas classes ou mais num mesmo elemento/tag do HTML, isso é possível e para fazer basta digitar, ao lado classe atual e depois de dar um espaçamento, o nome da próxima classe que deseja inserir. Exemplo:


        <h1 class="principal avançado">CSS Avaçado</h2>

Também é possível colocar um "id" e uma "class" num mesmo elemento/tag do HTML, para fazer isso é só abrir uma class com o seu respectivo nome logo após o "id" que já existe ou vise versa. Exemplo:


        <h1 id="principal" class="destaque">Criando Sites com HTML e CSS</h1>

Observação: Quando "id" e "class" estiverem juntos num mesmo elemento/tag, as configurações do "id" irão sempre sobrepor as configurações do "class" de uma mesma propriedade de estilo na hierarquia. A configuração mais específica sempre irá sobrepor a mais genérica.

Pseudo-classes

As pseudo-classes são relativas a um elemento ou uma classe. Uma pseudo-classe é uma palavra-chave adicionada às declarações de um seletor. Elas são relacionadas ao estado de um determinado elemento. No CSS, as Pseudo-classes são representadas por "dois pontos" (:).

Para usar as funcionalidades de Pseudo-classes, no código CSS, devemos abrir um seletor personalizado para o elemento que queremos determinar um estado, digitando "dois pontos" ( : ) e especificando o estado desejado, abrindo chaves posteriormente.

Alguns exemplos de estados de elementos (pseudo-classes) são: ativo, marcado, vazio, habilitado e etc.

Por exemplo, podem ser usadas para:

  • Estilizar um elemento quando um usuário move o mouse sobre ele
  • Estilize links visitados e não visitados de forma diferente
  • Estilize um elemento quando ele receber foco
  • Estilo de elementos de formulário válidos/inválidos/obrigatórios/opcionais

O estado (pseudo-classe) "hover" significa: "quando eu passar o cursor por cima", e permite que o elemento fique estilizado apenas quando estamos com o cursor por cima dele. No exemplo abaixo, tudo o que estiver dentro de "div" terá um background marrom quando o cursor estiver em cima do elemento:


        div:hover {
        background-color: brown;
    }

A pseudo-classe "visited" é bastante usada em âncoras/links, ela serve para ativar uma configuração de estilo quando uma unidade de elemento for visitada. Já a pseudo-classe "link" é um estado de elemento que serve para ativar uma configuração de estilo quando a unidade de elemento ainda não foi visitada.

As pseudo-classes em CSS são:

  • :active: Seleciona o link ativo
  • :hover: Seleciona links ao passar o mouse
  • :link: Seleciona todos os links não visitados
  • :visited: Seleciona todos os links visitados
  • :checked: Seleciona cada elemento <input> marcado
  • :disabled: Seleciona cada elemento <input> desabilitado
  • :empty: Seleciona cada elemento <p> que não tem filhos
  • :enabled: Seleciona cada elemento <input> habilitado
  • :first-child: Seleciona todos os elementos <p> que são primeiro filho de seu pai
  • :first-of-type: Seleciona cada elemento <p> queoprimeiro elemento <p> de seu pai
  • :focus: Seleciona o elemento <input> que tem foco
  • :in-range: Seleciona elementos <input> com um valor dentro de um intervalo especificado
  • :invalid: Seleciona todos os elementos <ingt; com um valor inválido
  • :lang(language): Seleciona cada elemento <p> com um valor de atributo lang começando com "it"
  • :last-child: Seleciona todos os elementos <p> que são o último filho de seu pai
  • :last-of-type: Seleciona cada elemento <p> que é o último elemento <p> de seu pai
  • :not(selector): Seleciona todos os elementos que não são um elemento <p>
  • :nth-child(n): Seleciona cada elemento <p> que é o segundo filho de seu pai
  • :nth-last-child(n): Seleciona cada elemento <p> que é o segundo filho de seu pai, contando a partir do último filho
  • :nth-last-of-type(n): Seleciona cada elemento <p> que é o segundo elemento <p> de seu pai, contando a partir do último filho
  • :nth-of-type(n): Seleciona cada elemento <p> que é o segundo elemento <p> de seu pai
  • :only-of-type: Seleciona cada elemento <p> que é o único elemento <p> de seu pai
  • :only-child: Seleciona cada elemento <p> que é o único filho de seu pai
  • :optional: Seleciona elementos <input> sem atributo "obrigatório"
  • :out-of-range: Seleciona elementos <input> com um valor fora de um intervalo especificado
  • :read-only: Seleciona elementos <input> com um atributo "readonly" especificado
  • :read-write: Seleciona elementos <input> sem atributo "readonly"
  • :required: Seleciona elementos <input> com um atributo "obrigatório" especificado
  • :root: Seleciona o elemento raiz do documento
  • :target: Seleciona o elemento #news ativo atual (clicado em um URL que contém esse nome de âncora)
  • :valid: Seleciona todos os elementos <input> com um valor válido

Pseudo-elementos

Os pseudo-elementos podem agir nas classes, nos elementos, nos id's e etc. Eles mexem diretamente no conteúdo periférico do elemento. No CSS, os pseudo-elementos são representados por dois pontos duplos (::)

Suponha que queremos que todo link no final tenha uma indicação de que é um link. Podemos abrir um seletor personalizado para links e usar o pseudo-elemento "::after" para fazer isso, como no exemplo:.


        a::after {
            content: ' ⇦';
            color: darkgray;
            font-weight: normal;
    }

O código acima faz com que manipulemos o conteúdo periférico posterior (pseudo-elemento "after") de links. A propriedade de estilo "content" detém o valor que foi determinado para aparecer logo em seguida de todo link que inserirmos no site (seta para esquerda). Além disso, as outras propriedades de estilo do exemplo configuram o estilo de cor e peso desse conteúdo periférico posterior.

Também existe o pseudo-elemento "before", que funciona como o "after" mas de forma oposta, fazendo com que manipulemos o conteúdo periférico que antecede os elementos.

Nota: Também conseguimos configurar pseudo-elementos e mexer em conteúdos periféricos quando usamos classes (class) . Exemplo:


    .especial::before {
        content: '⇨ ';
        color: darkgray;
        font-weight: normal;
        }

    .especial::after {
        content: ' ⇦';
        color: darkgrey;
        font-weight: normal;
        }

O código no exemplo acima utiliza pseudo-elementos para manipular conteúdo periférico de links que pertencem à classe (class) "especial".
