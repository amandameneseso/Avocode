Trabalhando com links

Usando links externos

Toda vez que clicamos em um local para ir para outra página e acessamos um site, estamos interagindo com um hyperlink. Para criar um hyperlink, devemos criar âncoras através da tag <a>. Exemplo:

<a href="https://google.com.br/">Google</a>

O atributo "hreflang" permite indicar o idioma principal do site para onde o link direciona.

O atributo "target" permite definir onde exibir o link. O valor "_blank" abre o link em uma nova aba ou janela. O valor "_self" abre o link na janela atual.

O atributo "rel" indica a natureza do destino. Aceita vários valores:

  • next: indica que o link é para a próxima parte do documento atual
  • prev: indica que o link é para a parte anterior do documento atual
  • author: indica que é um link para o site do autor do artigo atual
  • external: indica que o link é externo

Usando links internos

É possível também acessar um link interno no site. O arquivo do link deve estar na mesma pasta que o link principal. Exemplo:

Esta é a <a href="pag002.html" rel="next">segunda página</a> deste site.

Esta é a segunda página deste site.

Para criar link interno de um arquivo em uma sub-pasta, basta informar o caminho. Para se referir à pasta atual do servidor, pode usar ./ antes do nome do arquivo. Se quiser se referir à pasta imediatamente superior na hierarquia, use ../ para voltar um nível para a pasta mais externa.

Usando links para download

Para criar um link para download, basta fazer o link diretamente para o arquivo utilizando a tag <a>, o atributo "download", com o valor configurado para o nome do arquivo a ser baixado, e o atributo "type" para indicar ao navegador que tipo de arquivo está sendo baixado. Exemplo:

<li><a href="PDF 10 - Ligações em toda parte.pdf" download="PDF 10 - Ligações em toda parte" type="application/pdf">Baixar a aula em PDF</a></li>
        <li><a href="meulivro.zip" download="meulivro.zip" type="application/zip">Baixar arquivo em ZIP</a></li>

  • Baixar a aula em PDF
  • Baixar arquivo em ZIP

Atributo type: Especifica o media type do documento. Lista de media types.

Obs.: No Google Chrome, quando não é adicionado o atributo download, o link exibirá o arquivo em uma nova aba ou janela. Quando adicionado o atributo download, o arquivo será baixado. Também é possível utilizar o atributo target="_blank" para abrir o arquivo em uma nova janela.
