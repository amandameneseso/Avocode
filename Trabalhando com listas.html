Trabalhando com listas

Lista ordenada

A tag <ol> (ordered list) cria uma lista ordenada e a tag <li> (list item) identifica cada item da lista.

  1. acordar
  2. tomar café
  3. escovar os dentes

A tag <ol> possui um parâmetro type, onde configuramos o tipo de marcador da lista.

  1. acordar
  2. tomar café
  3. escovar os dentes

Também é possível indicar o início da contagem usando o parâmetro start.

  1. acordar
  2. tomar café
  3. escovar os dentes

Lista não ordenada

Para criar uma unordered list, vamos usar a tag <ul> para delimitar a lista e a tag <li> para criar cada item da lista.

  • pão
  • leite
  • tomate

O marcador padrão é a bolinha preta totalmente preenchida (disc), mas existe a opção de configurar a propriedade type para os seguintes valores: circle e square.

  • pão
  • leite
  • tomate

Dica: é possível criar vários itens ao mesmo tempo digitanto ol ou ul + li*(número de itens). Exemplo: ul > li*4.

Misturando listas

É possível aninhar listas, colocando a tag <ol> dentro de outra tag <ol>. Exemplo:

Minhas linguagens favoritas

  1. Antigas:
    1. Clipper
    2. Visual Basic
    3. Fortran
  2. Novas:
    1. PHP
    2. Python
    3. JavaScript

Além de aninhar listas ordenadas, podemos juntar listas <ol> com <ul> e vice-versa. Exemplo:

Meus jogos favoritos

  1. Nintendinho
    • Mario Bros
      • Super Mario Bros 3
      • Mario: Lost Levels
    • Ninja Gaiden
      • Ninja Gaiden 3
  2. Super Nintendo
    • Super Mario World
    • Donkey Kong
      • Donkey Kong Country 3
      • Diddy's Kong Quest
  3. PlayStation 1
    • Final Fantasy
      • Final Fantasy VII
      • Final Fantasy IV
    • Castlevania
      • Symphony of the Night

Lista de Definições

A tag <dl> (definition list) engloba uma lista de pares de termos e descrições. Um uso comum para este elemento é para implementar um glossário ou exibir metadados (uma lista de pares chave e valor). Cada termo é um <dt> (definition term) e cada descrição é um <dd> (definition description). Exemplo:

HTML
Linguagem de marcação para a criação do conteúdo de um site.
CSS
Linguagem de marcação para a criação do design de um site.
JavaScript
Linguagem de programação para a criação de interatividade de um site.
