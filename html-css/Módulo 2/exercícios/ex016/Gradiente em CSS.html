Testando gradiente

Podemos gerar gradientes e aplicarmos a componentes visuais usando folhas de estilo. Um gradiente é considerado pelo navegador como se fosse uma imagem, por isso usamos a propriedade background-image na declaração CSS. Para criar um gradiente usamos a função linear gradient. Também é possível gerar gradientes radiais, utilizando a função radial-gradient. Também é importante utilizar o parâmetro background-attachment: fixed se desejar que o degradê fixe na janela inteira.

O primeiro parâmetro da função indica o ângulo de inclinação (para indicar 90 graus, utilizamos "90deg" ou "to right") e as seguintes indicam as cores do degradê a ser criado. Você pode indicar quantas cores quiser.


        <style>
            body {background-image: linear-gradient(rgb(255, 255, 255), rgb(160, 202, 147), rgb(208, 148, 166)); background-attachment: fixed;}
        </style>
